# NetAudit — Network Security Auditor

Built by: Ahmed Al-Ghamdi | Network Engineer
Stack: Python 3 · Socket · Threading · HTML/CSS/JS

## What It Does
- Discovers live hosts via ping sweep
- Scans for open ports (dangerous + safe)
- Identifies weak protocols (Telnet, FTP, SMB, RDP)
- Calculates risk score per host (0-100)
- Generates JSON reports + interactive web dashboard

## Quick Start
python scanner.py 192.168.1.0/24
